JavaScript基础 Part1

Reference：https://www.ruanyifeng.com/blog/2011/04/quicksort_in_javascript

在有一点编程基础的前提下，短时间内学习一门新语言并非一件遥不可及的事，在本文中，我们对JavaScript做一个简单了解，并且使用一个快速排序函数来了解JavaScript的基础机制。

先贴出代码，这段快速排序代码摘自阮一峰的网络日志：

var quickSort = function(arr) {
　　if (arr.length <= 1) { return arr; }
　　var pivotIndex = Math.floor(arr.length / 2);
　　var pivot = arr.splice(pivotIndex, 1)[0];
　　var left = [];
　　var right = [];
　　for (var i = 0; i < arr.length; i++){
　　　　if (arr[i] < pivot) {
　　　　　　left.push(arr[i]);
　　　　} else {
　　　　　　right.push(arr[i]);
　　　　}
　　}
　　return quickSort(left).concat([pivot], quickSort(right));
};

Section1 变量声明与初始化:

var position; // 关键字 var 之后紧跟着的就是一个变量名，我们可以在初始化时为变量赋值
var position = 2021; // 初始化+赋值
var left = []; // var关键字同样用于初始化一个数组
var right = [2020, 2021]; // Array初始化+赋值
right[1]; // 2021，通过index索引数组的元素，从0开始

Section2 运算符:

js中有两种表示相等关系的运算符

  1. ==表示值相等
  2. ===表示值相等且数据类型相等，比如一个字符串和一个整型在数据类型上不等
hi = 2021 // 赋值
hi !== 2020; // true
hello = "2021"; //this is a string
hello == 12; // true! compare value
hello === 12; // false! compare type+value

Section4 函数:

声明一个函数也和声明一个变量一样，使用 var somefunction = something 的格式。在 JavaScript 之中，函数和变量本质上没有区别

var quickSort = function (myArray) {
    // do something about quick sort
    return 2021;
}
// 2021为这个函数的返回值，所以和其他语言一样，js的函数也可以当作变量传入其他的函数之中，这也是quicksort这个递归函数的基础

Section5 JavaScript的分号:

编写c语言程序时，必须使用;来进行不同语句之间的换行，在python中则不需要。在JavaScript中，使用;换行是一个可选项，但依然有可能造成误解，如下所示

var a = 3;
var b = 4;
var c = a+b;
(a+b).toString()

"7"

在上面的代码片段中，执行toString()前的代码使用;进行了换行，程序运行正确

var a = 3;
var b = 4;
var c = a+b //No delimiter 
(a+b).toString()

而上述这段代码执行错误，程序以为(a+b)的圆括号和上一行的 f 组成了一个函数调用，为了能让上述代码解析为两条不同的语句，我们就必须手动填写行尾的显式分号

如果想最大限度地避免因代码解析语句造成的错误的话，规范的使用;依旧是个好选择
