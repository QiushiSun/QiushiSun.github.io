Node.js Introduction

Node.js是web编程相关实验的JavaScript的运行环境，本篇文章对其做一个简要的介绍

Node.js是一个Javascript运行环境(runtime environment)，实质是对Chrome V8引擎进行了封装。Node.js对一些特殊用例进行优化，提供替代的API，使得V8在非浏览器环境下运行得更好。 V8引擎执行Javascript的速度非常快，性能非常好。Node.js是一个基于Chrome JavaScript运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。Node.js 使用事件驱动， 非阻塞I/O模型而得以轻量和高效，非常适合在分布式设备上运行数据密集型的实时应用。(Source:wikipedia)

简而言之：Node.js是一种JavaScript的运行环境，能够使JavaScript的运行脱离浏览器

安装地址如下：Node.js-Download

安装完成后，可以使用node --version 查看node.js的版本，以检查是否安装成功

Node.js的主要特点

  1. Node.js主要用于Web服务器一类的网络应用，Node.js与其他编程语言环境最大的不同之处在于，PHP等语言是阻塞的（顺序执行，只有前一条命令执行完毕才会执行后面的命令），而Node.js是非阻塞（Non- blocking）的（即多条命令可以并行，通过回调函数(callback)得知命令已结束运行）。
  2. Node.js是事件驱动的。开发者可以在不使用线程的情况下开发出一个能够承载高并发的服务器。

Node.js使用Google V8 JavaScript引擎，特点如下(Source:wikipedia)：

  • V8是基于BSD许可证的开源软件
  • V8速度非常快
  • V8专注于网络功能，在HTTP、DNS、TCP等方面更加成熟

Node.js的大量功能模块可以通过一个名为npm的包管理器下载

What is npm?

npm是随Node.js安装的包管理工具，能解决node.js代码部署上的问题

  • npm允许用户从服务器下载别人编写的第三方包安装到本地使用。
  • npm允许用户从服务器下载并安装别人编写的命令行程序到本地使用。
  • npm允许用户将自己编写的包或命令行程序上传到npm服务器供别人使用。

从Node.js官方网站安装好Node.js后，可以使用npm list -g查看所有安装的模块
