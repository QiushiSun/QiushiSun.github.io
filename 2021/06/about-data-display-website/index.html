About Data Display Website

延续之前的爬虫数据采集项目，搭建一个数据展示网站，根据课程要求，网站主要实现以下几个功能

Basic Requirements:

  1. 实现登录注册界面，非注册用户无法直接访问
  2. 注册、登陆、查询等操作全部记入数据库中的日志（此部分使用MongoDB和MySQL配合完成）
  3. 爬虫数据查询结果列表支持分页和排序
  4. 用可视化工具（Echarts和D3）实现数据分析图表展示在网站中
  5. 实现一个管理端界面，可以查看用户操作以及注销用户

Advanced Requirements:

  1. 实现中文分词查询
  2. 实现查询结果按照主题词（关键字）的排序
  3. 使用Elastic Search+Kibana展示爬虫数据结果

本项目的博客依旧展示于我基于Hugo搭建的博客：项目博客地址：Qiushisun

Remark: 数据展示网站所包含的内容

  1. 延续之前的爬虫项目所爬取的新闻数据（网易，新浪，中国新闻）
  2. 新爬取的豆瓣电影数据（包括电影名，导演，演员，上映时期，以及影评）

整合一下上述需求，将其分为以下几个板块进行展示

  1. 用户登录、注册与加密
  2. 管理员权限与管理员控制面板
  3. 注册、登陆、查询等操作记入数据库中的日志
  4. 各类数据查询的分页展示与排序，基于查询主题词/星数的排序
  5. 不同参数组合的进阶数据查询方式
  6. 基于D3和Echarts的数据可视化（中期已经使用Echarts做过简单的可视化，本次丰富了可视化的形式，除了词云外，还实现了渐变图，动态图，地图热度等进阶效果）
  7. 中文分词查询
  8. 使用Elastic Search+Kibana进行数据展示
