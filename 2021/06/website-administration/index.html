Section 1: 实现管理员权限

管理员界面和普通用户界面的区别就在于管理员多了一个用户管理页面可以对用户信息做管理+添加新数据。此部分请浏览manager组件部分，实现的逻辑并不困难，admin是注册时候确定的，也可以在数据库中更改权限

同样是依靠token来实现，如果没有token的话，我们就返回验证错误。如果有token有效，就先验证这个token的合法性，是就传递，否则返回Invalid token.

const jwt = require("jsonwebtoken");
const config = require("config");

module.exports = function(req, res, next) {
  if (!config.get("requiresAuth")) return next();

  const token = req.header("x-auth-token");
  if (!token) return res.status(401).send("Denied.");
  try {
    const decoded = jwt.verify(token, config.get("jwtPrivateKey"));
    req.user = decoded;
    next();
  } catch (ex) {
    res.status(400).send("Invalid Token.");
  }
};

Section 2: 用户管理

管理员可以对用户账号进行管理

Part1：用户信息查看

我们首先需要使用 getuserlist来获取相应的信息

然后通过Api.get方法可参照 axios官方文档 。 此处的 config 是为请求提供的配置信息，直接利用了axios.get,在请求头中加入token，也就是利用

headers: { Authorization: `Bearer ${token}` },

后端验证了这个token，然后把response的内容，利用setState方法更新本地state，管理员就可以获得这个项目中用户的表单信息

具体方法如下

getUserList() { // 获取当前用户信息
  const token = Util.getToken();
  Api.get('/users/list', {
    headers: { Authorization: `Bearer ${token}` }, // tokens验证
  }).then((res) => {
    if (res.data) {
      this.setState({
        userListData: res.data.data,
      });
    }
  });
}

Part2：用户信息删除

实现一个账号删除功能，同样需要进行提示，以免产生误操作导致用户被消失

// 实现用户删除功能，跳出一个警告框
deleteUser = (item) => {
    const _this = this;
    confirm({
      // 以下为确认框中的信息，要给一点提示防止把删除的对象搞错...
      title: 'Are you sure to do this?',
      content: 'You are deleting user!',
      okText: 'confirm',
      okType: 'danger',
      cancelText: 'cancel',
      onOk() {
        const token = Util.getToken();
        const postUrl = `/users/${item._id}/delete`;
        Api.post(
          postUrl,
          {},
          {
            headers: { Authorization: `Bearer ${token}` },
          },
        )
          .then((res) => {
            console.log(res);
            if (res.data) {
              // 成功后返回相应的提示，如果错误则需要抛出异常
              message.success('Deleted successfully');
              _this.getUserList();
            } else {
              message.error('fail');
            }
          })
          .catch((err) => {
            console.log(err);
          });
      },
      onCancel() {
        console.log('Cancel');
      },
    });
  };
