Web Programming: 项目总结与思考

Data Display Website

Web 编程项目开发总结

功能复盘：

最后是本次项目的开发经验总结，先简单进行一下功能复盘

项目的数据源为新浪新闻、网易新闻以及豆瓣影评，与中期无异。事实上如可视化，排序和搜索功能在期中项目的时候就已经实现了，在后半段的开发历程中主要是利用了React框架，对原有的功能做了进一步的整合和升级，完成了本次项目所要求的所有的基本要求和所有的扩展要求。

项目功能分条陈述如下

  1. 用户登录、注册与加密：这部分主要实现了用户注册—登录—登出，以及对用户密码进行了加密处理
  2. 管理员权限：这部分主要实现了管理员设置，用户管理，数据增删改（如增删改电影信息）等功能
  3. 日志信息、错误处理与地理位置：此处实现了日志管理，用户登录/注册地位置记载，以及各类错误情况跳转
  4. 分页与查询进阶：此部分注重网页的美观化，设置了每页条目个数与页数跳转，再进行不同参数组合的查询实现
  5. 分词查询与打分排序：此处注重使用中文分词查询与基于关键词的排序查询
  6. 数据可视化：这部分扩展了期中项目，分别使用了Echarts和D3实现了各类数据分析图表与词云，包括渐变图，动态图，地图热度等进阶效果
  7. 基于Elastic Search + Kibana的结果展示

总结与感悟：

如开头展示的思维导图和上述条目所示，本次项目的开发把课程的一些要求做了整合，然后加入了一些自己感兴趣的额外功能，并利用React框架进行开发。

内容分别从思维导图中的这六个方面进行开发经历展示。首先是完成了用户登录认证、用户注册，信息加密（基于crypto-js），再实现了用户/数据管理，添加favorites等管理员权限。接着基于课堂上的知识，完成了日志信息处理与错误处理，并尝试了用户地理位置获取（基于morgan和Geolocation ）。在数据展示这一块，美化了期中项目的分页机制，和页面跳转机制，增加了不同关键词搭配的查询方式以及分词查询，借助Echarts和D3（以及wiki地图数据和百度地图api），拓展了期中的可视化功能，进行了多种可视化结果展示包括了，饼状图，环状图，词云，折线，地图热度等各类可视化方法，最后还借助了Elastic Search + Kibana进行了数据展示。

Web编程课程让我收获颇多，本学期在这门课程上的工作量也还算比较大，不过在开发过程中仍然发现自己对框架和JS的掌握程度还不够高（之前的科创项目和其他大作业都是在用Django和Flask进行开发）。经过这门课程的学习和训练，学会了大量在日后JavaScript开发中需要使用的前后端技巧，并熟悉了一些框架的使用，也努力尝试了通过写博客这种方式详细阐述一个项目的完整开发流程，讲解了一些技术的要点、技术的原理、如何借鉴已有工作以及我认为可以有进一步应用/改进的地方，而非一味复制粘贴代码或是灌水。
